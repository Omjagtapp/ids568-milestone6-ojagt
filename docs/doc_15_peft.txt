Title: Parameter-Efficient Fine-Tuning (PEFT)

PEFT methods fine-tune a small fraction of parameters while freezing the pre-trained backbone. LoRA (Low-Rank Adaptation) adds trainable rank-r matrices to weight updates: W' = W + BA where B is in R^(d x r) and A is in R^(r x k). Prefix tuning prepends soft tokens to the input. Adapters insert small bottleneck modules between transformer layers. PEFT is critical for fine-tuning large models on consumer hardware because it drastically reduces GPU memory requirements. QLoRA combines LoRA with 4-bit quantisation, enabling fine-tuning of 65B parameter models on a single 48GB GPU.
